ABOUT THE PROJECT

FoodNation is an interactive World map that let you explore top dishes of each country, Find recipes tutotials ,and restaurant finder for those delicious cuisines.

This project was made with HTML,CSS3,Javascript(ES6),JSON,PHP,and MySQl

APIs Used: Google GeoCharts, Google Maps, YouTube, Yelp
